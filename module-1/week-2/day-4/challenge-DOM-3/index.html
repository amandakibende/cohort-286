Pokemons !

Instructions

An array of pokemons will be provided
We want to be able to display a list of pokemons and
add a filter functionality in order to search for pokemons by their names

  1. Display the pokemons name on your page
  2. Add an input with an id
  3. Set an event handler on the input (oninput)
  4. Get the input value and filter the pokemons
  5. Display the filtered list on your page

BONUS

When you click on a pokemon you should be able to see
the selected pokemon. Have a look at the find() method and data-attributes
